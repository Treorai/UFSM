## Regressão Linear

Prever um valor numérico contínuo  

Em termos algébricos, $y=mx+b$


$y' = b+w_1x_1$

$y'$ ou $\hat{y}$ : rótulo previsto, ou seja, a saída  
$b$ : bias do modelo  
$w_1$ : peso  
$x_1$ : feature value  

### Exemplo

$y' = b + w_1x_1 + w_2x2 + w_3x3 + ... w_nx_n$
