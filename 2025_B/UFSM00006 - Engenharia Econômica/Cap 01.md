# Juros

## Definição das grandezas

- Valor presente / Capital / Principal ( $P$ ) [R$]
- Valor futuro / Montante ( $F$ ) [R$]
- Tempo ( $n$ ) [a.p.: ao período]
- Juros ( $J$ ) [R$]
- Taxa de juros ( $i$ ) [% a.p.]

## Fórmulas

- $i=\frac{J}{P}$
- $F=P+J$

Simples
- $J=i \cdot P \cdot n$
- $F=P(1+i \cdot n)$

$$i= \frac{(\frac{F}{P}-1)}{n}$$

Composto
- $J=P((1+i)^n -1)$
- $F=P(1+i)^n$

$$i= \sqrt[n]{ \frac{F}{P}} -1$$
